# คู่มือการเริ่มต้นใช้งานระบบโรงเรือนอัตโนมัติ

## สิ่งที่ต้องเตรียม

### อุปกรณ์ฮาร์ดแวร์

1. **Arduino Uno** (1 บอร์ด)
2. **Sensor วัดความชื้นในดิน** (Soil Moisture Sensor) (1 ตัว)
3. **Relay 4-Channel** แบบ Active-Low (1 โมดูล)
4. **ปั๊มน้ำ DC** พร้อมท่อน้ำ (1 ตัว)
5. **พัดลม DC** (1 ตัว)
6. **LCD Display 16x2 (I2C)** สำหรับแสดงผล (1 จอ)
7. **สายไฟจัมเปอร์** (หลายเส้น)
8. **สาย USB Type-B** สำหรับต่อ Arduino กับคอมพิวเตอร์
9. **แหล่งจ่ายไฟ** สำหรับปั๊มน้ำและพัดลม

### ซอฟต์แวร์ที่ต้องติดตั้ง

- **Arduino IDE** (โปรแกรมสำหรับเขียนและอัปโหลดโค้ด)
- **LiquidCrystal I2C Library** (ไลบรารีสำหรับควบคุม LCD)

## 🔌 ขั้นตอนที่ 1: ต่อสายวงจร

### 1.1 ต่อ Sensor วัดความชื้นในดิน

| ขา Sensor | ต่อกับ Arduino |
| --------- | -------------- |
| VCC       | 5V             |
| GND       | GND            |
| AO        | A0             |
| DO        | ไม่ต้องต่อ     |

### 1.2 ต่อ Relay 4-Channel (12V)

| ขา Relay (Input) | ต่อกับ...         | รายละเอียด                              |
| ---------------- | ----------------- | --------------------------------------- |
| VCC              | Arduino 5V (+)    | ต่อขั้วบวกจาก Arduino 5V                |
| JD-VCC           | แบตเตอรี่ 12V (+) | **ห้ามต่อ +12V กับ Arduino โดยตรง**     |
| GND              | แบตเตอรี่ 12V (-) | **Arduino GND ต่อเข้า Sensor เท่านั้น** |
| IN1              | Arduino D2        | (สำรอง)                                 |
| IN2              | Arduino D3        | (สำรอง)                                 |
| IN3              | Arduino D4        | ควบคุมปั๊มน้ำ                           |
| IN4              | Arduino D5        | ควบคุมพัดลม                             |

### 1.3 ต่อ LCD Display 16x2 (I2C)

| ขา LCD | ต่อกับ Arduino | รายละเอียด              |
| ------ | -------------- | ----------------------- |
| VCC    | 5V             | ไฟเลี้ยง 5V             |
| GND    | GND            | กราวด์                  |
| SDA    | A4             | สาย Data (I2C)          |
| SCL    | A5             | สาย Clock (I2C)         |

> [!TIP]
> **หมายเหตุ**: ที่อยู่ I2C ของ LCD มักจะเป็น `0x27` หรือ `0x3F` ขึ้นอยู่กับรุ่น
> ถ้าไม่แน่ใจ สามารถใช้โปรแกรม I2C Scanner เพื่อตรวจสอบที่อยู่ได้

### 1.4 ต่อปั๊มน้ำและพัดลม

- **ปั๊มน้ำ**: ต่อเข้า Relay ช่อง 3 (IN3)
- **พัดลม**: ต่อเข้า Relay ช่อง 4 (IN4)

> [!WARNING]
> ระวังเรื่องขั้วไฟ และควรใช้แหล่งจ่ายไฟแยกสำหรับปั๊มน้ำและพัดลม

## ขั้นตอนที่ 2: ติดตั้ง Arduino IDE

### สำหรับ Windows

1. ไปที่เว็บไซต์ https://www.arduino.cc/en/software
2. คลิก **"Windows Win 10 and newer, 64 bits"**
3. คลิก **"Just Download"** (ไม่ต้องบริจาคก็ได้)
4. เปิดไฟล์ที่ดาวน์โหลดมา แล้วทำตามขั้นตอนการติดตั้ง
5. คลิก **"I Agree"** > **"Next"** > **"Install"**
6. รอจนติดตั้งเสร็จ แล้วคลิก **"Close"**

### สำหรับ macOS

1. ไปที่เว็บไซต์ https://www.arduino.cc/en/software
2. คลิก **"macOS"**
3. คลิก **"Just Download"**
4. เปิดไฟล์ .dmg ที่ดาวน์โหลดมา
5. ลาก Arduino IDE ไปใส่โฟลเดอร์ Applications

### สำหรับ Linux

1. ไปที่เว็บไซต์ https://www.arduino.cc/en/software
2. คลิก **"Linux AppImage 64 bits"**
3. เปิด Terminal แล้วพิมพ์:
   ```
   chmod +x arduino-ide_*_Linux_64bit.AppImage
   ./arduino-ide_*_Linux_64bit.AppImage
   ```

## ขั้นตอนที่ 3: ติดตั้ง LiquidCrystal I2C Library

### 3.1 เปิด Library Manager

1. เปิดโปรแกรม **Arduino IDE**
2. คลิก **Tools** > **Manage Libraries...** (หรือกด `Ctrl + Shift + I`)
3. หน้าต่าง Library Manager จะเปิดขึ้นมา

### 3.2 ค้นหาและติดตั้ง Library

1. ในช่องค้นหา พิมพ์ **"LiquidCrystal I2C"**
2. มองหา Library ชื่อ **"LiquidCrystal I2C"** โดย **Marco Schwartz**
3. คลิกที่ Library แล้วคลิกปุ่ม **"Install"**
4. รอจนติดตั้งเสร็จ (จะแสดงคำว่า "INSTALLED")
5. คลิก **"Close"** เพื่อปิดหน้าต่าง

### 3.3 ตรวจสอบการติดตั้ง

1. คลิก **File** > **Examples**
2. เลื่อนลงมาจะเห็น **LiquidCrystal I2C** ในรายการ
3. ถ้าเห็นแสดงว่าติดตั้งสำเร็จแล้ว

### 3.4 (ถ้าต้องการ) ตรวจสอบที่อยู่ I2C ของ LCD

1. ดาวน์โหลดโค้ด **I2C Scanner** จาก Arduino Examples
2. คลิก **File** > **Examples** > **Wire** > **i2c_scanner**
3. อัปโหลดโค้ดไปยัง Arduino (ต่อ LCD ไว้แล้ว)
4. เปิด **Serial Monitor** (`Ctrl + Shift + M`)
5. ดูที่อยู่ I2C ที่แสดงผล (เช่น `0x27` หรือ `0x3F`)
6. นำที่อยู่นี้ไปใช้ในโค้ด (ถ้าไม่ใช่ `0x27` ให้แก้ไขในไฟล์ main.cpp บรรทัดที่ 33)

```cpp
constexpr uint8_t LCD_I2C_ADDRESS = 0x27;  // เปลี่ยนเป็น 0x3F ถ้าใช้ที่อยู่นี้
```

## ขั้นตอนที่ 4: เปิดโค้ดใน Arduino IDE

### 4.1 เปิดโปรแกรม Arduino IDE

ดับเบิลคลิกที่ไอคอน Arduino IDE บนเดสก์ท็อปหรือในเมนูโปรแกรม

### 4.2 สร้างไฟล์ใหม่

1. คลิก **File** > **New Sketch** (หรือกด `Ctrl + N`)
2. จะมีหน้าต่างใหม่เปิดขึ้นมา

### 4.3 คัดลอกโค้ด

1. เปิดไฟล์ [main.cpp](https://github.com/Scott-Nx/Automatic-Greenhouse-System/blob/main/main.cpp) ในโปรเจกต์นี้
2. เลือกทั้งหมด (กด `Ctrl + A`)
3. คัดลอก (กด `Ctrl + C`)
4. กลับไปที่ Arduino IDE
5. ลบโค้ดเดิมทั้งหมด แล้ววาง (กด `Ctrl + V`)

### 4.4 บันทึกไฟล์

1. คลิก **File** > **Save** (หรือกด `Ctrl + S`)
2. ตั้งชื่อว่า **"GreenhouseSystem"**
3. เลือกที่เก็บไฟล์ แล้วคลิก **Save**

## ขั้นตอนที่ 5: เชื่อมต่อ Arduino กับคอมพิวเตอร์

### 5.1 เสียบสาย USB

ใช้สาย USB Type-B เชื่อมต่อ Arduino Uno กับคอมพิวเตอร์

### 5.2 เลือกบอร์ด

1. คลิก **Tools** > **Board** > **Arduino AVR Boards** > **Arduino Uno**

### 5.3 เลือก Port

1. คลิก **Tools** > **Port**
2. เลือก Port ที่มีคำว่า **"Arduino Uno"** ต่อท้าย
   - Windows: จะเป็น `COM3`, `COM4`, หรือ `COM` อื่นๆ
   - macOS: จะเป็น `/dev/cu.usbmodem...`
   - Linux: จะเป็น `/dev/ttyACM0` หรือ `/dev/ttyUSB0`

> [!TIP]
> **ถ้าไม่เห็น Port**: ลองถอดสาย USB แล้วเสียบใหม่ หรือลองเปลี่ยนช่อง USB

## ขั้นตอนที่ 6: อัปโหลดโค้ดไปยัง Arduino

### 6.1 ตรวจสอบโค้ด (Verify)

1. คลิกปุ่ม **✓ (เครื่องหมายถูก)** ที่มุมซ้ายบน หรือกด `Ctrl + R`
2. รอจนแสดงข้อความ **"Done compiling."** ที่ด้านล่าง
3. ถ้ามีข้อผิดพลาด จะแสดงเป็นตัวอักษรสีส้ม/แดง

### 6.2 อัปโหลดโค้ด (Upload)

1. คลิกปุ่ม **→ (ลูกศร)** ที่อยู่ถัดจากปุ่ม Verify หรือกด `Ctrl + U`
2. รอจนแสดงข้อความ **"Done uploading."**
3. ไฟ LED บน Arduino จะกะพริบระหว่างอัปโหลด

## ขั้นตอนที่ 7: ดูผลการทำงาน

### 7.1 ดูผลจอ LCD Display

หลังจากอัปโหลดโค้ดเสร็จแล้ว LCD จะแสดงผล:

**หน้าจอเริ่มต้น (2 วินาทีแรก):**
```
🌱 Greenhouse
System v2.1
```

**หน้าจอแสดงสถานะ:**
```
🌱M: 65% OK
IDLE       123s
```

**คำอธิบายการแสดงผล:**
- **แถวที่ 1**: 
  - ไอคอน (🌱=ปกติ, 💧=รดน้ำ, 🌀=ระบาย, ⚠=ผิดพลาด)
  - `M:XX%` = ค่าความชื้นเป็นเปอร์เซ็นต์
  - สถานะ (`OK`=ปกติ, `DRY`=แห้ง, `WET`=ชื้นเกิน)
- **แถวที่ 2**:
  - สถานะระบบ (`IDLE`, `WATERING`, `VENT`, `COOLDOWN`)
  - เวลาที่ผ่านไปในสถานะปัจจุบัน (วินาที)

### 7.2 เปิด Serial Monitor

1. คลิก **Tools** > **Serial Monitor** หรือกด `Ctrl + Shift + M`
2. หน้าต่างใหม่จะเปิดขึ้นมา

### 7.3 ตั้งค่า Baud Rate

ที่มุมขวาล่างของ Serial Monitor ให้เลือก **"9600 baud"**

### 7.4 อ่านค่า

จะเห็นข้อความแสดงค่าความชื้นและสถานะของระบบ เช่น:

```
=====================================
ระบบโรงเรือนอัตโนมัติ เริ่มทำงาน
Automatic Greenhouse System Started
=====================================
เริ่มต้นระบบสำเร็จ!

ค่าความชื้น (Moisture): 450 | สถานะ: ปกติ (NORMAL)
  ปั๊มน้ำ: ปิด | พัดลม: ปิด
```

## ขั้นตอนที่ 8: ปรับแต่งค่า (ถ้าต้องการ)

### ค่าที่สามารถปรับได้

เปิดไฟล์ [main.cpp](https://github.com/Scott-Nx/Automatic-Greenhouse-System/blob/main/main.cpp) แล้วหาบรรทัดต่อไปนี้:

```cpp
#define MOISTURE_DRY_THRESHOLD    700   // ค่าขีดจำกัดดินแห้ง
#define MOISTURE_WET_THRESHOLD    300   // ค่าขีดจำกัดดินชื้นเกินไป
#define PUMP_RUN_TIME             5000  // เวลาเปิดปั๊มน้ำ (มิลลิวินาที)
#define FAN_RUN_TIME              10000 // เวลาเปิดพัดลม (มิลลิวินาที)
```

### วิธีปรับค่า

| ค่า                      | ความหมาย                  | คำแนะนำ                                  |
| ------------------------ | ------------------------- | ---------------------------------------- |
| `MOISTURE_DRY_THRESHOLD` | ค่าที่ถือว่าดินแห้ง       | ยิ่งเพิ่มค่า = ดินต้องแห้งมากถึงจะรดน้ำ  |
| `MOISTURE_WET_THRESHOLD` | ค่าที่ถือว่าดินชื้นเกินไป | ยิ่งลดค่า = ดินต้องชื้นมากถึงจะเปิดพัดลม |
| `PUMP_RUN_TIME`          | เวลาเปิดปั๊มน้ำ           | 5000 = 5 วินาที                          |
| `FAN_RUN_TIME`           | เวลาเปิดพัดลม             | 10000 = 10 วินาที                        |

### วิธีหาค่าที่เหมาะสม

1. เปิด Serial Monitor ดูค่าความชื้น
2. ทดสอบจุ่ม Sensor ในดินแห้ง จดค่าไว้
3. ทดสอบจุ่ม Sensor ในดินชื้น จดค่าไว้
4. ปรับค่า `THRESHOLD` ตามที่ต้องการ

## การแก้ปัญหาเบื้องต้น

### ปัญหา: LCD ไม่แสดงผล

**อาการ**: จอ LCD ไม่มีอะไรแสดง หรือแสดงแต่กล่องสี่เหลี่ยม

**วิธีแก้**:

1. **ตรวจสอบการต่อสาย**:
   - VCC ต่อกับ 5V
   - GND ต่อกับ GND
   - SDA ต่อกับ A4
   - SCL ต่อกับ A5

2. **ตรวจสอบ Backlight**:
   - หมุนไปมาที่ตัวปรับความสว่าง (potentiometer สีน้ำเงินด้านหลัง LCD)
   - ถ้าเห็นแสงสว่างแต่ไม่มีตัวอักษร ให้หมุนไปอีกทาง

3. **ตรวจสอบที่อยู่ I2C**:
   - ใช้โค้ด I2C Scanner ตรวจสอบที่อยู่ (ดูขั้นตอนที่ 3.4)
   - ถ้าที่อยู่ไม่ใช่ `0x27` ให้แก้ไขในโค้ด

4. **ตรวจสอบ Library**:
   - ตรวจสอบว่าติดตั้ง LiquidCrystal I2C Library แล้ว
   - ลองถอดแล้วติดตั้งใหม่

### ปัญหา: LCD แสดงตัวอักษรแปลกๆ (Garbled Text)

**อาการ**: LCD แสดงตัวอักษรที่อ่านไม่ออก

**วิธีแก้**:

1. หมุนตัวปรับความสว่าง (Contrast Potentiometer) ด้านหลัง LCD
2. ตรวจสอบสาย SDA และ SCL ว่าต่อถูกต้อง
3. ตรวจสอบว่าไม่มีสายอื่นรบกวนสัญญาณ I2C

### ปัญหา: I2C Scanner ไม่พบ LCD

**อาการ**: I2C Scanner แสดง "No I2C devices found"

**วิธีแก้**:

1. ตรวจสอบการต่อสาย SDA (A4) และ SCL (A5)
2. ตรวจสอบว่า LCD ได้รับไฟ (VCC, GND)
3. ลองใช้สายจัมเปอร์เส้นอื่น
4. ตรวจสอบว่าโมดูล I2C บน LCD ไม่เสีย

### ปัญหา: อัปโหลดไม่ได้

**อาการ**: แสดงข้อความ "avrdude: stk500_recv(): programmer is not responding"

**วิธีแก้**:

1. ตรวจสอบว่าเลือก Board เป็น "Arduino Uno"
2. ตรวจสอบว่าเลือก Port ถูกต้อง
3. ถอดสาย USB แล้วเสียบใหม่
4. ลองเปลี่ยนสาย USB

### ปัญหา: ค่าความชื้นไม่เปลี่ยน

**อาการ**: ค่าแสดงเป็น 0 หรือ 1023 ตลอด

**วิธีแก้**:

1. ตรวจสอบการต่อสายของ Sensor
2. ตรวจสอบว่า VCC ต่อกับ 5V, GND ต่อกับ GND
3. ตรวจสอบว่าสาย AO ต่อกับ A0

### ปัญหา: Relay ไม่ทำงาน

**อาการ**: ไม่ได้ยินเสียงคลิกจาก Relay

**วิธีแก้**:

1. ตรวจสอบการต่อสาย VCC และ GND ของ Relay
2. ตรวจสอบว่าสาย IN1-IN4 ต่อถูกขา
3. ตรวจสอบว่า Relay ได้รับไฟเพียงพอ

### ปัญหา: ปั๊มน้ำ/พัดลม ไม่ทำงาน

**อาการ**: Relay คลิก แต่ปั๊มน้ำ/พัดลมไม่หมุน

**วิธีแก้**:

1. ตรวจสอบการต่อสายปั๊มน้ำ/พัดลมกับ Relay
2. ตรวจสอบแหล่งจ่ายไฟของปั๊มน้ำ/พัดลม
3. ลองทดสอบปั๊มน้ำ/พัดลมโดยตรงกับแหล่งจ่ายไฟ
